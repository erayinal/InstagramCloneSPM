# InstagramCloneSPM
A clone of the instagram application written in swift 5 and using firebase 5 as a backend server.

## Used Technologies
* Firebase
* Cocoapods
* Swift Package Manager(SPM)
* CoreData
* Tab Bar Controller
* Tableview Controller
